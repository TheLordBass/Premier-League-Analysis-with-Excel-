# Premier-League-Analysis-with-Excel-

An Interactive Performance Tracker Built Entirely in Microsoft Excel

Project Overview
This project transforms raw Premier League match data into a dynamic, user-friendly dashboard. By leveraging advanced Excel functions and data modelling techniques, it provides a comprehensive look at team standings, player form, and match-day statistics for the current season.

Technical Features
Dynamic Data Modelling: Utilised Power Query to clean and transform raw CSV data into a structured relational model.

Advanced Analytics: Implemented complex formulas (XLOOKUP, INDEX/MATCH, and Nested IFS) to calculate live league standings and Goal Difference (GD).

Interactive UI: Designed a professional-grade interface using Slicers, Timelines, and Conditional Formatting for seamless navigation between clubs.

Automated Reporting: Built a "One-Click Refresh" system to update the entire dashboard as new match results are added.

Key Insights
Performance Tracking: Visualises "Home vs. Away" win probabilities across all 20 clubs.

Team Deep-Dive: A dedicated selector to compare Manchester-based clubs against the rest of the league.
